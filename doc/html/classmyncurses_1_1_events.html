Detailed Description

Class containing the ncurses events handling. This class must NOT be copied. You MUST use only ONE Events object by Window (undefined behavior).

◆ termKeypad() [1/2]

int myncurses::Events::termKeypad ( bool  bf) const

The keypad option enables the keypad of the user's terminal. If enabled (bf is TRUE), the user can press a function key (such as an arrow key). If disabled (bf is FALSE), curses does not treat function keys specially and the program has to interpret the escape sequences itself. The default value for keypad is FALSE.

Parameters
bfBoolean used to turn-on/off the keypad.
Returns
Return the integer OK on success and ERR on failure.
